---

### 📦 Requirements

- Inside `<main>` after a `<form>` tag create a `<section>` element

- Inside the `section` add:
    - `<img>` — user profile image
    - `<h2>` — user name
    - `<p>` — email
    - `<p>` — skills
    - `<p>` — description
    - `<a>` — message link

---
### 🧩 Structure rules

Group content logically using `<div>`:
1. Profile header
    - image
    - name
    - email
2. Profile content
    - skills
    - description
3. Action area
    - message link
