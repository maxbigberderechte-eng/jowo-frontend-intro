# 🟠 HTML Task 2 — Form Structure

### 🎯 Goal
Create a form to collect user data.

### 📦 Requirements
- Inside `<main>` create a `<form>`
- Add `<h2>` with text `Create profile` heading inside `<form>`
- Add the following `<input>` fields:
    - name
    - email
    - skills
    - about
- Each field must include:
    - `label` connected with `input`
    - `input` with correct `type`
    - `name` attribute
    - `id` attribute
    - `placeholder` text
- All fields must be required
- Add buttons:
    - Submit button
    - Reset button

--- 

### 👀 Check yourself
- Can you type into all fields?
- Does clicking on a label focus the input?
- Can you submit an empty form?
- Does the email field require a valid email format?
- Does the Reset button clear all fields?
