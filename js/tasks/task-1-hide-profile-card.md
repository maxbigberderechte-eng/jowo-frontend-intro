# 🟡 JS Task 1 — Hide profile card

### 🎯 Goal
Learn how to select an element and add a class using JavaScript.

---

### 📦 Requirements

- Create a file called `app.js`
- Select the profile card element using class
- Add a class `hidden` to it using JavaScript

---

### 💡 Hint
Make sure you have this class in CSS:

```css
.hidden {
  display: none;
}
```
