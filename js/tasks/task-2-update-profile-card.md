# ЁЯЯб JS Task 2 тАФ Update Profile Card

### ЁЯОп Goal
Learn how to update DOM elements using form values and handle form events.

---

### ЁЯУж Requirements
- Select the form element using `.class`
- Select form inputs using `#id`
- Select card fields (`name`, `email`, `skills`, `about`) using `.class`
- Update profile card when form is submitted
- Hide profile card when form is reset
- Show profile card when data is submitted


#### On submit:
- Prevent page reload
- Read values from inputs:
  - name
  - email
  - skills
  - about
- Update profile card content
- Remove class `hidden` from `.profile-card`

---

#### On reset:
- Add class `hidden` to `.profile-card`

---

### ЁЯза Important

You will use:

- `querySelector`
- `addEventListener`
- `event.preventDefault()`
- `textContent`
- `innerHTML`
- `classList.add()`
- `classList.remove()`
